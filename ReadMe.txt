Weather Alert Visualization System
A Python based system for managing, visualizing, and analyzing weather alerts.

Overview
The Weather Alert Visualization System simulates how real world emergency systems manage 
and prioritize weather alerts. It allows users to add alerts, process them in order, 
filter results, and visualize severity levels in a clear and structured format.

Features
Adding weather alerts with type, severity, location, and message
Visualizing severity using a dynamic bar (█ and ░)
Emoji based alert type representation
Queue system (FIFO) for processing alerts
Undoing feature using stack (LIFO)
Filtering alerts by type or severity
Sorting alerts by severity (high → low)
Tracking alert types by region
Generating summary statistics

Installation
Requirements
Python 3.0 or higher

Steps
Download or copy the file

Save the file as:

weather_alert_system.py
Open terminal / command prompt / tool

Run the program:

python weather_alert_system.py

Limitations
Console based (no GUI)
Uses static sample data (no real time weather API)
Limited predefined alert types
